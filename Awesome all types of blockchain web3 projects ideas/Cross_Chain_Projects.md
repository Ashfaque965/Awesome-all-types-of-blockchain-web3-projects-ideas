# What is Cross-Chain?

**Cross-chain technology** enables communication, asset transfers, and data sharing between independent blockchain networks.

Instead of operating in isolation, multiple blockchains can exchange tokens, messages, smart contract calls, and application state through interoperability protocols.

---

# Why Cross-Chain?

* Interoperability
* Liquidity Sharing
* Multi-Chain Applications
* Asset Portability
* Better User Experience
* Blockchain Composability
* Ecosystem Expansion
* Reduced Vendor Lock-In

# Common Interoperability Models

## Asset Bridges

Move tokens between blockchains.

Examples

* Wrapped Assets
* Lock-and-Mint
* Burn-and-Mint

---

## Cross-Chain Messaging

Transmit arbitrary messages between blockchains.

Examples

* Contract Calls
* Governance Messages
* Data Synchronization

---

## Shared Security

Multiple chains inherit security from another blockchain.

---

## Interoperability Networks

Dedicated interoperability protocols connecting many blockchains.

---

## Atomic Swaps

Trust-minimized peer-to-peer token exchanges across different blockchains.
