# What is a Smart Contract?

A **smart contract** is a self-executing program deployed on a blockchain. Once deployed, its code and state are managed by the blockchain, enabling transparent and deterministic execution.

## Characteristics

* Decentralized execution
* Immutable deployment (unless designed to be upgradeable)
* Cryptographically secured
* Transparent state transitions
* Programmable business logic
